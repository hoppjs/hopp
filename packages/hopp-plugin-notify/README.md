# hopp-plugin-notify

![There was a demo here.](.github/demo.gif)

Shows push notifications on build completion. Useful for watch tasks.

## Usage

Simple:

```javascript
export default
  hopp('src/*.js')
    .notify()
    .dest('dist')

export const watch =
  hopp.watch([
    'default'
  ])
```

All [node-notifier](https://www.npmjs.com/package/node-notifier) options are valid.
